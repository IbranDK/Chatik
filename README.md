# Chatik 
Многопользовательский TCP-чат на основе сокетов

## Описание 
Chatik — это десктопное приложение для обмена текстовыми сообщениями в локальной сети.
Система реализована на C# с WPF-интерфейсом и построена на архитектуре клиент-сервер с многопоточной обработкой соединений и событийной моделью.

## Основные возможности 
- Подключение нескольких клиентов одновременно
- Broadcast-рассылка сообщений всем участникам чата
- Личные сообщения между пользователями (`/pm <имя_пользователя> <сообщение>`)
- Список пользователей онлайн в реальном времени
- Уведомления о входе и выходе участников
- Лог событий сервера с временными метками

## Технологии 
| Технология | Назначение |

| C# / .NET 8 | Основной язык и платформа |

## Как запустить 

### Требования
- Visual Studio 2022
- .NET 8
- Windows
- Оба компьютера в одной локальной сети

### Запуск сервера
1. Открыть решение `Chatik.sln` в Visual Studio
2. Запустить проект **ProjectServer**
3. Нажать кнопку «Запустить»
4. Узнать свой IP адрес:
Win+R
ipconfig

### Запуск клиента
1. Запустить проект **ProjectClient**
2. Ввести IP сервера, порт `5000` и никнейм
3. Нажать «Подключиться»
